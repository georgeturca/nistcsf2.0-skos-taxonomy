# nistcsf2.0-skos-taxonomy
Research thesis repository for creating a machine-readable SKOS taxonomy of the NIST Cybersecurity Framework 2.0.

This repository contains the research artifacts for the thesis project:

**Creating a Machine-Readable SKOS Taxonomy of the NIST Cybersecurity Framework 2.0**

## Goal

The goal of this project is to represent the NIST (CSF) 2.0 as a machine-readable SKOS taxonomy.

## Planned Artifacts

- RDF/SKOS taxonomy of the NIST CSF 2.0
- Structured extraction file
- SPARQL competency questions
- SHACL validation shapes
- Initial mapping to an existing ISO/IEC 27001 SKOS taxonomy
- Web-based taxonomy browser
